### Developing

This resume is written in LaTeX and builds with xelatex.

```bash
# Install required system packages
sudo apt-get install -y \
    lmodern \
    pgf \
    texlive-font-utils \
    texlive-fonts-recommended \
    texlive-latex-base \
    texlive-latex-extra \
    texlive-xetex \
    xzdec

# Build the resume
./build
```
